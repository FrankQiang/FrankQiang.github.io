#### zip遍历两个迭代器

```python
    name = ["John", "Tom", "Json"]
    count = [81, 94, 78]
    
    for name, count in zip(name, count):  # 惰性生成
        print(name, count)

    # John 81
    # Tom 94
    # Json 78
```

#### 最短的耗尽，zip就会终止

```python
    name = ["John", "Tom", "Json", "Dave"]
    count = [81, 94, 78]
    
    for name, count in zip(name, count):
        print(name, count)

    # John 81
    # Tom 94
    # Json 78
```
